 # Desktop 为 PC 桌面端
 * 出牌时采用鼠标点击策略
 # Android 为安卓手机端
 * 出牌时采用触摸连续拖动策略
 
 # 编译环境
 1. 开发框架：Felgo（配置为 Qt 5.15）
 2. 编译工具：qmake
 3. 编译器：Qt for Android（Android Clang）
